Tic Tac Toe (AI)

Using Java I recreated Tic Tac Toe, the game has its own GUI that was created using JFrame, you click on the interface to place X on the grid. You play against an AI that places O's moves on the grid. The placing of the AI's moves is dependant on yours it can also detect winning moves for itself and favors those against defending. I find most users I have test the program ending up in a Tie or Loss!
